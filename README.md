# Flickr Gallery
The most recent version of this page can be found [here](http://flickr-gallery.netlify.com).

## Built with
  * VanillaJS
  * Flickr API
  * Promise poly-fill
  * Fetch API

## Setup
To get it up and running open a terminal in the root folder and run:

``` shell
npm start
```
